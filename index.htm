tipos de teste:

Teste de caixa preta:

Teste de caixa-preta é um teste de software para verificar a saída dos dados usando entradas de vários tipos. Tais entradas não são escolhidas conforme a estrutura do programa. Quanto mais entradas são fornecidas, mais rico será o teste.

Teste de Caixa branca

este de caixa-branca é uma técnica de teste que usa a perspectiva interna do sistema para modelar os casos de teste. No teste de software, a perspectiva interna significa basicamente o código fonte. No teste de hardware, cada nó de um circuito pode ser testado.

Teste de tarefa

é uma avaliação em que os participantes são solicitados a realizar uma série de atividades ou tarefas específicas relacionadas a uma habilidade, conhecimento ou competência em particular. Esse tipo de teste é comumente usado em diversos contextos, como recrutamento e seleção de pessoal, avaliação educacional, avaliação de habilidades técnicas, entre outros.
